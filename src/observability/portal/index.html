Not yet instrumented

This portal can only show what lerobot-eval's stock output already contains. It cannot show:

  • Per-step inference latency (how long each select_action call took)
  • Chunk boundaries (when a new action chunk was generated vs. which env steps consumed it)
  • Action staleness (how old the observation behind an executed action was by the time it ran)
  • Denoising-step-level timing inside the reverse-diffusion loop
  • Raw per-step state (agent/block pose) outside of decoding video frames by eye
  • Discarded predictions (the actions a chunk generated but never executed)
  • Per-step reward curve or divergence point for a failed episode
  • Compute utilization (CPU/MPS/memory) during a rollout
  • Sim-time vs. policy-inference-time split per step
  • Pre/post-processing overhead separate from model forward-pass time
  • Per-episode latency variance (only aggregate eval_s / eval_ep_s exist)
  • Statistical confidence on a success rate (no variance across seeds/N)
